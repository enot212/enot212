PHP и JavaScript

Языки программирования PHP и JavaScript позволяют сделать сайт динамичным, то есть реагирующим на действия пользователя.

Виды скриптов

Скрипты — программы, позволяющие реагировать на действия пользователя. Бывают серверные (PHP) и клиентские (JavaScript).
